这是一个仿照侦探小说术语的笔记模版（whodunit、howdunit和whydunit）

## 怎样学习一个知识
### Whydunit (Why done it?)
为什么需要它 / 产生背景 / 解决什么问题

### Whatzit (What is it?)
它是什么 / 核心概念 / 底层原理

### Howdunit (How done it?)
【最简答的】怎么使用它 / 语法 / API

### Watchit (Watch out for it)
注意事项 / 常见坑点 / 易错点

### Wheredunit (Where done it?)
应用领域 / 使用场景

### Worthit (Is it worth it?)
优缺点 / 性能权衡 / 替代方案
